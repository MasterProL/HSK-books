# HSK Books

A collection of HSK Chinese learning materials organized by level from HSK 1 through HSK 6.

## Repository Overview

This repository contains study resources and book materials for the Hanyu Shuiping Kaoshi (HSK) Chinese proficiency exams. The content is organized into separate folders for each level:

- `HSK 1`
- `HSK 2`
- `HSK 3`
- `HSK 4`
- `HSK 5`
- `HSK 6`

## Purpose

The repository is designed to help Chinese learners access HSK study books and related resources in a simple, level-based structure.

## How to Use

1. Open the folder for the HSK level you are studying.
2. Browse the available materials and practice content.
3. Use the files to support your HSK exam preparation.

## Notes

- Keep the folder structure intact for easier navigation.
- If you add more materials, place them in the folder matching the HSK level.
